This repository is an independently evolving project built from a larger
refactor and redesign effort based on the author's earlier cloud-disk codebase.

## Current Position

- `netdisk_v1` is the new mainline project.
- The original cloud-disk repository remains a reference source and legacy
  implementation source.
- Code that was reused from the earlier project has been copied into this
  repository and progressively merged into the new module layout.
